# CMS指纹字典
## 搜集国内/外常见CMS指纹
____
> 有提交字典的，可以提issue或者发我
> 字典慢慢更新
> 部分指纹上传之后会更新一个多线程扫描脚本

### CMS 格式
+ `cms` cms名字, 英文小写
+ `file_type` 文件类型
+ `md5` md5值
+ `path` 路径
+ `type` 类型
+ `uptime` 更新时间/增添时间
```json
{
    "cms": "dedecms",
    "file_type": "css",
    "type": "md5/keyword/re"
    "md5": "[md5值]",
    "path": "/public/static/css/style.css",
    "uptime": "2019-05-21 19:03:43"
}
```
