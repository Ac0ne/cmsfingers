### CMS 格式
+ `type` 文件类型
+ `type_size` 文件大小
+ `md5` md5值
+ `path` 路径
+ `uptime` 更新时间/增添时间
```json
{
    "type": "css",
    "file_size": 2.4375,
    "md5": "[md5值]",
    "path": "/public/static/css/style.css",
    "uptime": "2019-05-21 19:03:43"
}
```
